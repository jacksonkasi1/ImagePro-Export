## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/imagepro-export.git
   ```

2. Navigate to the project directory:

   ```bash
   cd plugin
   ```

3. Install the dependencies:

   ```bash
   bun install
   ```

4. Run the application:

   ```bash
   bun dev
   ```
